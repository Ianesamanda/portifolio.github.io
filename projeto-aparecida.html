APARECIDA DO NORTE

3º Semestre - Projeto Interdisciplinar

I. Descrição e Tecnologias

Este sistema foi desenvolvido para resolver a administração logística de viagens de ônibus de uma paróquia com destino a Aparecida do Norte. O sistema informatiza a escolha de coordenadores, o controle de poltronas/vagas, e o registro de pagamentos e reservas, substituindo os antigos controles manuais.

A arquitetura do projeto foi separada em duas frentes tecnológicas robustas:

  • Back-end (API): Construído em Java com Spring Boot. Utiliza PostgreSQL como banco de dados relacional, migrações com Flyway, e controle de autenticação seguro via Spring Security e tokens JWT.
  • Front-end: Interface de usuário intuitiva desenvolvida com HTML5, CSS3 e JavaScript puro, consumindo os endpoints da API REST.

II. Minha Participação
