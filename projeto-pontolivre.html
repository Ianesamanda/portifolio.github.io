6º Semestre - Projeto Interdisciplinar.

I. Descrição e Tecnologias

Este projeto consiste na recriação mobile do clássico jogo de raciocínio lógico "Campo Minado". A aplicação exige precisão matemática para gerar aleatoriamente os perigos pelo tabuleiro e calcular a proximidade das minas em relação aos blocos adjacentes.

A arquitetura do jogo baseia-se em:

  • Gerenciamento de Estado Avançado: Uso de hooks do React Native para controlar a grade de blocos, o status de vitória/derrota e a pontuação do usuário.
  • Lógica Algorítmica: Implementação de matrizes (arrays 2D) e funções recursivas (para revelar áreas seguras encadeadas).
  • Plataforma: Desenvolvido e testado em tempo real no ambiente Expo Snack.

II. Minha Participação

Este Mini Projeto foi desenvolvido integralmente por mim. Fui a única responsável por toda a esteira de criação, desde a estruturação algorítmica do "motor" do jogo (espalhamento das minas e cálculo de vizinhança) até a construção da interface gráfica (UI) utilizando os componentes nativos do React Native.
